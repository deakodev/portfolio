## Overview

MedStudForty is a full-stack web platform designed to improve diagnostic reasoning in preclinical medical students preparing for USMLE Step 1 and COMLEX Level 1 board exams. The platform combines a curated question bank, user-generated content tools, performance analytics, and personalized tutoring services.

## Key Features

### Real-Time Test Emulation

Exam-authentic environment replicating the actual NBME/NBOME testing experience with high-fidelity tooling:

- Built-in calculator for dosage and clinical calculations
- Fuzzy search lab value reference for quick lookup
- Question flagging and review to mark uncertain answers
- Persistent timer with pause/resume functionality
- User feedback forms embedded for reporting improvements
- Fully responsive with light/dark themes

### Content Management System

- Dashboard for creating and editing custom questions
- Rich text editor with support for images and formatting
- User-generated question library with version control
- Moderation tools for quality assurance

### Performance Analytics

- Session tracking across question blocks
- Score comparisons with peer cohorts
- Progress visualization over time
- Weak area identification for targeted review

![Example Test Results Dashboard](</(medstudforty)/msf_results.jpeg>)

### Personalized Tutoring

- One-hour consultation service for action planning
- Data-driven review strategies based on performance metrics
- Individualized study schedules aligned to exam blueprints

## Architecture

**Frontend:** TypeScript + React with Next.js for server-side rendering and routing

**Backend:** RESTful API with Next.js API routes

**Database:** MySQL with Prisma ORM for type-safe queries

**Authentication:** Secure user sessions with JWT tokens

**Deployment:** Vercel with continuous integration

**Additional Features:**

- SEO optimization for discoverability
- Analytics integration for user behavior insights
- Discord community integration for peer support
